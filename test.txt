Valid Directory com absolute path:
/home/jomunoz   =   "bash: /home/jomunoz: Is a directory"
$? = 126

Valid File com absolute path:
/home/.../minishell/ficheiro   =  "bash: /home/.../minishell/ficheiro: Permission denied"
$? = 126
